Route Mapper

RouteMapper

Description

The goal with RouteMapper was to try to re-create some of the features I like about MapMyRide and Strava, with an added bonus I've always wanted of being able to see an overview of bike rides I've done plotted on top of each other.

This turned into a little-mapping-app-that-could as I struggled with the Google Maps, Elevation and Visualization APIs to achieve the desired functionality. The Maps API lets you define a small set of waypoints for a route to pass through, but I wanted to be able to create precise routes for arbitrarily long bike rides. This led to a change from draggable routes, to an implementation that only ever asks the Maps API for a route between two points. Stubbornly, I then manually modify the response to include the previous route state.

This app features from-scratch authentication storing encrypted passwords, ActiveRecord associations supporting followed user relationships, JSON-serialized map data and a Backbone.js frontend backed up by a Rails API.

Minesweeper

Minesweeper

Description

This is a pretty true-to-form command line implementation of Minesweeper. Choose a grid size, then navigate with the W-A-S-D keys. Press R to reveal a tile, and F to flag a bomb.

The trick with this game was figuring out the recursive algorithm to check a neighboring tile for bombs before revealing squares. Once I had this working, I went back and changed the user input from a prompt for coordinate pairs, to the current more fluid version supporting keyboard navigation on a seemingly static grid. The last touch was allowing a user to serialize their current game to JSON to load and play later.
